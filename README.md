# 📊 Sales Performance Analysis

## 📌 Business Problem
The goal of this project is to analyze sales data to identify trends, top-performing products, and opportunities to increase revenue.

---

## 📊 Project Overview
This project uses Python to clean, analyze, and visualize sales data, transforming raw data into actionable insights for decision-making.

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

---

## 📈 Key Analysis

- Sales by category
